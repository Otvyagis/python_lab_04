# Python_lab_4

Решение для варианта 15.

## Что реализовано
 - Генератор, циклически перебирающий список городов и выдающий 1 000 000 значений.
 - Функция-генератор, создающая кортежи случайных чисел.
 - Преобразование пользовательской строки в таблицу 3×N, используя zip.
 - Реализован графический интерфейс
 - Все ключевые функции покрыты тестами (pytest)

 ## Требования
- Python 3.8+
- PySide6>=6.2
- pytest

## Как запускать
 - python main.py
